# Grocery Shop Management App

A console-based inventory management system for grocery stores with role-based access control.

## Tech Stack
![Java](https://img.shields.io/badge/Java-8+-orange)
![JDBC](https://img.shields.io/badge/JDBC-Connectivity-blue)
![MySQL](https://img.shields.io/badge/MySQL-4479A1)

## Features
- Role-based access: Manager vs Receptionist
- Billing management and invoice generation
- Stock/inventory tracking
- Employee record management
- Automated sales reporting

## Setup Instructions
1. Clone repository
2. Open in NetBeans IDE
3. Configure MySQL database
4. Run Main.java
